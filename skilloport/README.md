# SkillOport Frontend

This folder now contains the React/Vite frontend only.

## Run

Start the backend separately on port `3000`, then run:

```bash
npm install
npm run dev
```

Vite serves the frontend and proxies `/api` to `http://localhost:3000`.

## Related folders

- `../skilloport-backend` — standalone Java/Spring Boot backend for IntelliJ
- `../skilloport-node-legacy` — archived old Node backend files

# Readrop Frontend

React/Vite frontend for the Readrop free book giveaway platform.

## Run

Start the backend separately on port `3000`, then run:

```bash
npm install
npm run dev
```

Vite serves the frontend and proxies `/api` to `http://localhost:3000`.

## Related folders

- `../readrop-backend` — standalone Java/Spring Boot backend for IntelliJ
